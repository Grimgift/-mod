# More tea leaf

茶葉の種、生の茶葉、幾つかのお茶の葉とお茶を追加するMODです

茶葉の種は園芸店等で入手できます

茶葉の種を育てると生の茶葉を収穫できます

制作にて紅茶の葉や緑茶、烏龍茶の葉にすることが出来ます

開発版8863にて作成、開発版8918でもエラーなどは出ないことを確認

作者が更新不能な状況で更新の必要がある場合は遠慮なく改変、公開をしてください
